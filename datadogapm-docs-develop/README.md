# Datadog手順書

本手順書はオープンIT基盤の案件において、APM機能をDatadogへ移行するためのDatadog導入手順および導入後のDatadogAPMの理解、ダッシュボード利用方法について記載する。

## 対象読者  
- オープンIT基盤の案件業務担当者が対象。インフラチームの作業は別途YSD運用技術部より連携される。

Datadog移行作業の概要は以下を参照。
